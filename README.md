## URL Shortener Built with Spring Boot
A fast and lightweight URL shortener that converts long links into short, shareable URLs with analytics, redirect support, and easy link management.
URL shortening service with Base62 short-code generation, persistent storage, analytics tracking, and RESTful management endpoints.

## How to run
```
mvn spring-boot:run
```
